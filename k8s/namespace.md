# namespace.yaml — пространство имён приложения

## Назначение
Создаёт namespace `apps`, в котором разворачиваются все сервисы Bird. Это упрощает
управление ресурсами и даёт единый контур для сетевых политик.

## Состав ресурсов
- `Namespace apps`

## Что и зачем указано
- `metadata.name: apps` — имя пространства имён, которое используется во всех остальных манифестах.
- `metadata.labels.app.kubernetes.io/part-of: bird` — метка для группировки ресурсов проекта.

## Что менять
- Обычно менять ничего не нужно.
- Если требуется другое имя namespace, его нужно синхронно обновить во всех остальных манифестах.

## Как проверить
```
kubectl get ns apps
kubectl get all -n apps
```
